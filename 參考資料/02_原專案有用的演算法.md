## 1. Chlamy 細胞濃度公式(球狀細胞)

```python
def chlamy_concentration(cell_boxes, total_area, cell_volume=271.8, depth=0.1):
    avg_radius = np.mean(list(map(lambda box: (abs((box[3] - box[1]) / 2) +
                                                abs((box[2] - box[0]) / 2)) / 2, cell_boxes)))
    expected_radius = (cell_volume * 3 / 4 / math.pi) ** (1/3)
    return len(cell_boxes) / (((expected_radius ** 2) / (avg_radius ** 2) * total_area)
                              * 10 ** -9 * depth)
```

**邏輯:**
- 假設細胞是球狀,標準體積 `cell_volume`(預設 271.8 μm³,Chlamydomonas typical)
- 由體積反推「期望半徑」`expected_radius`
- 從 bounding box 算「實際平均半徑」`avg_radius`
- 用 (expected/actual)² 校正放大率 → 推算實際視野的物理面積
- 細胞數 ÷ (面積 × 深度 × 1e-9) → cells/mL

**跟我們現在的 step6 公式差別:**
- step6 直接用「25 張 = 1 microliter」的固定體積
- 原版用「細胞物理大小 + 視野面積 + 深度」反推體積
- 兩種都對,只是假設不同
- **如果換顯微鏡或鏡頭**,原版公式會自動補正,我們的需要重新校正

## 2. Diatom 細胞濃度公式(長條狀)

```python
def diatom_concentration(cell_boxes, image_area, cell_length, mm_depth):
    avg_length = np.mean(list(map(lambda x: ((x[2] - x[0]) ** 2 + (x[3] - x[1]) ** 2) ** 0.5, cell_boxes)))
    return len(cell_boxes) / (((cell_length / avg_length) ** 2 * image_area)
                              * 10 ** -9 * mm_depth + 1e-9)
```

跟 chlamy 類似,但用「長度」而不是「半徑」(因為 diatom 是長條形)。

## 3. 三種成長曲線回歸(細胞培養趨勢分析)

時間序列(t1 = 0 hr 拍一次,t2 = 24 hr 拍一次,...)→ 看細胞濃度怎麼長:

### Linear(線性成長 — 短期)
```python
z = np.polyfit(x, y, 1)   # y = a*x + b
poly1d_fn = np.poly1d(z)
```

### Exponential(指數成長 — 對數期)
```python
def exponential_f(x, a, b, c):
    return a * np.exp(b*x) + c
(a, b, c), _ = optimize.curve_fit(exponential_f, x, y)
```

### Logistic(S 型曲線 — 完整培養週期)
```python
def logistic_f(x, a, b, c, d):
    return a / (1. + np.exp(-c * (x - d))) + b
(a, b, c, d), _ = optimize.curve_fit(logistic_f, x, y, method="trf")
```

**用途:** 拍幾天的細胞濃度曲線後,套用 logistic 可以推算:
- 最大濃度(carrying capacity, a + b)
- 對數期斜率(c)
- 半飽和時間(d)

非常有用的細胞培養分析工具。
